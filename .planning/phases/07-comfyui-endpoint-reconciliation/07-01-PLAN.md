**3. `last4(key: string): string` helper.** Returns `'****' + key.slice(-4)` if key.length >= 4 else `'****'`. Used for banner only.

**4. `fetchWithTimeout(url, init, timeoutMs)` helper.** Wraps `fetch` in `AbortController`; on timeout, throws `new Error('timeout after ${timeoutMs}ms')` (NOT AbortError — friendlier matrix cell).

**5. `discoverDocsBase(): Promise<string | null>` helper.** Implement verbatim from 07-RESEARCH.md §Landmine 5 lines 332-354 (the regex parser). Returns `null` on fetch error / 4xx/5xx / regex miss. Never throws.

**6. `probe(base, path, apiKey)` helper.** Signature: `async (base: string, path: string, apiKey: string) => Promise<{ base, path, status: number | 'ERR' | 'TIMEOUT', snippet: string, elapsedMs: number }>`. Uses `fetchWithTimeout`, calls `new URL(path, base)`, sends `headers: { 'X-API-Key': apiKey }`, `method: 'GET'`, `redirect: 'manual'` (per 07-PATTERNS.md §"Redirect policy" — X-API-Key leakage prevention).

On response: read body with `res.text()` capped at first 200 bytes via `(await res.text()).slice(0, 200)`. Apply a scrub: `snippet = snippet.replaceAll(apiKey, '[redacted]')` (Landmine 6 defense).

On network error or timeout: status = `'ERR'` or `'TIMEOUT'`, snippet = error message.

```typescript
const apiKey = process.env.COMFYUI_API_KEY;
if (!apiKey || apiKey.trim() === '') {
  console.error(ansi.red('FATAL:') + ' COMFYUI_API_KEY not set.');
  console.error(ansi.dim('  Ensure .env exists at repo root with COMFYUI_API_KEY=... and re-run.'));
  console.error(ansi.dim(`  Current cwd: ${process.cwd()}`));
  process.exit(3);
}

console.log(ansi.bold('── ComfyUI Endpoint Probe (Phase 7, D-EP-01..05) ──'));
console.log(`  cwd: ${process.cwd()}`);
console.log(`  key: ${last4(apiKey)}`);
console.log(`  timeout per probe: ${PROBE_TIMEOUT_MS}ms`);
console.log('');

// Discover docs-advertised base (graceful degradation).
const docsBase = await discoverDocsBase();
if (docsBase) {
  console.log(ansi.dim(`  docs-advertised base: ${docsBase}`));
} else {
  console.error(ansi.yellow('  note: docs fetch failed — proceeding with hardcoded bases only'));
}
console.log('');

// Build final base list (dedup docs vs hardcoded).
const bases = [...HARDCODED_BASES, ...(docsBase && !HARDCODED_BASES.includes(docsBase as typeof HARDCODED_BASES[number]) ? [docsBase] : [])];

// Run serial matrix.
const results: Array<Awaited<ReturnType<typeof probe>>> = [];
for (const base of bases) {
  for (const path of PROBE_PATHS) {
    const r = await probe(base, path, apiKey);
    results.push(r);
    const statusStr = typeof r.status === 'number'
      ? (r.status === 200 ? ansi.green(String(r.status)) : r.status === 401 ? ansi.yellow(String(r.status)) : ansi.red(String(r.status)))
      : ansi.red(String(r.status));
    console.log(`  ${statusStr}  ${base}${path}  ${ansi.dim(`(${r.elapsedMs}ms)`)}`);
    if (r.snippet) console.log(ansi.dim(`       ${r.snippet.slice(0, 100).replace(/\n/g, ' ')}`));
  }
}
```

## Trust Boundaries

| Boundary | Description |

| repo → ComfyUI Cloud | Outbound HTTPS; request bears `X-API-Key` header; response body treated as untrusted (may echo header) |

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-DRIFT-01 | Information Disclosure | scripts/probe-comfy-endpoint.mts | mitigate | Key is never passed to `console.log`. Banner emits `****${last4}` only (see `last4()` helper).
