| T-06-05-01 | Tampering | qNum accepts negatives that SQLite then silently clamps | mitigate (this plan) | Reject with HTTP 400 + INVALID_INPUT envelope BEFORE the engine call. | T-06-05-03 | Information Disclosure | Error message includes the raw `${raw}` input | mitigate | Per orchestrator threat-model preamble: hard-reject negatives + non-integer floats with HTTP 400 before any DB call. | T-06-05-04 | Repudiation | A bad request is not logged | accept | The typedErrorHandler shapes the 400 response; whether requests are logged is a transport-layer concern handled at server.ts (Hono logger middleware). Out of scope. |
